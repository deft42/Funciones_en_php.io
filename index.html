FUNCIONES Y FUNCIONES INCORPORADAS.

Definición de una función.

En programación, una función es un conjunto de instrucciones que realizan una tarea específica y pueden ser reutilizadas en diferentes partes de un programa. En PHP, las funciones se definen con la palabra clave function.

Declaración y sintaxis de una función basicas.

function nombreDeLaFuncion() {
    // Cuerpo de la función
		// Aqui van a ir todas las instrucciones que se van a realizar.
}

// Para poder accionar la función, se tiene que mandar a llamar.
// para esto solamente tenemos que escribirla por su nombre.
nombreDeLaFuncion();

Existen dos tipos de función, las cuales son.

Función sin parámetros: Este tipo de función no toma ninguna información cuando es llamada. Cuando una función no tiene parámetros, significa que no espera recibir ningún valor cuando es llamada. Por ejemplo:

// Ejemplo de una funcion sin parametros. Como podemos observar, dentro de los
// parentesis no hay ningun valor que se va a pasar.
function saludar() {
    echo "Hola, mundo! desde una funcion sin parametros!!";
}

saludar();
// Llamada a la función sin parámetros, en donde tambien podemos observar cuando
// se manda a llamar no se insertan parametros dentro de los parentesis.
//OUTPUT EN PANTALLA
Hola, mundo! desde una funcion sin parametros!!

Función con parámetros: Una función con parámetros está diseñada para recibir uno o más valores cuando es llamada. Dichos valores seran personalizados a la hora de ser llamados. Aquí hay un ejemplo:

// Ejemplo de una funcion con parametros. Como podemos observar, dentro de los
// parentesis hay o existe un valor que va tener un efecto dentro de la función.
function saludar_persona($nombre) {
    echo "Hola, $nombre!, esta es una funcion con parametros";
}

saludar_persona("Juan");
// Llamada a la función con parámetros, en donde tambien podemos observar existe
// un valor entre los parentesis. Dicho valor personaliza el
// lugar del parametro anterior.

En este caso, la función saludar_persona espera un parámetro $nombre y luego lo utiliza para personalizar el saludo. Al llamar a la función, le pasamos el valor "Juan", por lo que imprimirá "Hola, Juan!".

// OUTPUT EN PANTALLA
Hola, Juan!, esta es una funcion con parametros

Parámetros y valores de retorno.

  • Los parámetros son variables locales a la función y se definen entre los paréntesis de la declaración de la función.
  • Las funciones pueden devolver un valor con la palabra clave return.
  • return se utiliza para finalizar la ejecución de una función y devolver un valor. Cuando una función alcanza una declaración return, la función se detiene y el valor especificado después de return se devuelve al lugar desde donde se llamó a la función.
// Definimos una funcion que va a obtener dos parametros
// Parametro 1.- a.
// Parametro 2.- b.
function sumar($a, $b) {
		// La función va a tener la instruccion de realizar una operacion basica.
		// Va a sumar los dos paramertos obtenidos.
		// a + b
    return $a + $b;
		// Terminada la ejecucion de la operacion, vamos retornar el valor obtenido.
		// Resultado de a + b.
}

// Se manda a llamar a la funcion para activarla.
$resultado = sumar(2, 3);
// OUTPUT EN PANTALLA

// Interpretacion de la funcion "sumar" (a + b).

// Instruccion, suma -------> a + b
	// Termina de ejecutar la funcion y retorna los valores de a y b.

// Cuando se manda a llamar a la funcion por su nombre "sumar".
	// Personalizacion de valores 2 + 3

// Impresion del resutlado -> 5
	// Resultado en pantalla:
5

Variables locales:

Las variables locales son aquellas que se definen dentro de una función y solo son accesibles dentro de esa función. No pueden ser utilizadas fuera de la función ni son visibles para otras funciones.. Ejemplo:

// Declaramos una funcion sin parametros que se llamara "Saludar"
function saludar() {
		// Dentro la función se a declarado una variable que va a contener
		// una cadena de texto. 
    $mensaje = "¡Hola, mundo!";
		// Pasamos a realizar una impresion en pantalla de la cadena de texto.
    echo $mensaje;
}

// Se manda a llamar a la función para que realice su instrucción. En este caso
// como no tiene parametros no espera un valor para realizar una accion. Y 
// ejecutara por defecto la instruccion que tiene dentro de ella.
saludar();

// Si intentamos acceder a la variable desde afuera de la función, se nos va
// a regresar un mensaje de error, esto por que el programa al intentar
// encontrarla y no hacerlo, nos mostrara un mensaje de "variable no definida"

Variables globales:

Las variables globales son aquellas que se definen fuera de cualquier función y están disponibles en todo el script. Sin embargo, para acceder a una variable global dentro de una función, necesitas utilizar la palabra clave global.

// Se define una variable que va a contener una cadena de texto.
$mensajeGlobal = "¡Hola, mundo!, soy una variable global";

// Se crea una variable con el nombre "Saludar" la cual no va a tener
// ningun parametro.
function saludar() {
		// Se accesa desde adentro a la variable global con la palabra reservada
		// "global"
    global $mensajeGlobal;
		// Seguido pasamos a imprimir en pantalla lo que se encuentre dentro de la
		// variable "mensajeGlobal"
    echo $mensajeGlobal;
}

// Se manda a llamar a la funcion para que realize su accion.
// Como es una función sin parametros, no esta esperando ningun valor
// para realizar una tarea en especifico, por lo que se pasara a ejecutar
// las instrucciones que tenga dentro de ella.
saludar();
// OUTPUT EN PANTALLA 
¡Hola, mundo!, soy una variable global
La palabra reservada "global" en PHP se utiliza para acceder a variables globales desde dentro de una función. Cuando declaras una variable fuera de una función, esa variable tiene un ámbito global y no puede ser accedida directamente desde dentro de la función. Para superar esto, puedes usar la palabra reservada "global" para indicar que estás haciendo referencia a la variable global.

Funciones predefinidas isset & empty.

Son funciones en PHP que se utilizan para verificar si una variable está definida y si está vacía, respectivamente.

isset.

  • Se utiliza para comprobar si una variable está definida, es decir, si existe y no es nula.
  • Devuelve true si la variable existe y tiene un valor distinto de null, de lo contrario, devuelve false.

Puede ser util.

Verificar la existencia de variables: Esto es útil para asegurarte de que una variable exista antes de intentar utilizarla. Evita errores relacionados con variables no definidas.

$nombre = true;

// Verificamos si la variable existe o esta definida con una condición.
// Si la variable nombre, existe, entonces:
if (isset($nombre)) {
		// Imprime en pantalla el siguiente mensaje.
    echo "La variable \$nombre está definida.";
} else {
		// De no cumplir, imprime en pantalla el siguiente mensaje.
    echo "La variable \$nombre no está definida.";
}

// Podemos decir que una variable definida, es cuando tiene un valor asignado.
// No importa si es de tipo numerico, cadena de texto, funcion, etc.

//Ejemplos:
//$nombre = false; -------------------> Variable definida.
//nombre = 0; ------------------------> Variable definida.
//nombre ="Cadena de texto"; ---------> Variable definida.
//nombre; ----------------------------> Variable NO definida.

Manejo de parámetros en formularios:

Aquí, isset se utiliza para verificar si el parámetro 'nombre' ha sido enviado a través de un formulario antes de intentar usarlo.

$nombre = isset($_POST['nombre']) ? $_POST['nombre'] : 'Valor Predeterminado';
Si la condición es verdadera, la variable toma el valor después del ?.

Si la condición es falsa, la variable toma el valor después de :.

Comprobación de elementos en matrices o arreglos asociativos: 

// Creamos un arreglo asociativo. Asociativo por que tiene una clave - valor
// El arreglo tiene la cantidad de 2 elementos con sus respectivas clases.
	// Valor 1
	// Valor 2
$miArray = array('clave1' => 'valor1', 'clave2' => 'valor2');
// Realizamos la verificacion de la existencia de alguna clase que queramos.
// Si, el ID o clave con asignación "clave1" es verdadera / existe dentro de
// mi arreglo asociado llamado "miArray" realiza lo sigueinte.
if (isset($miArray['clave1'])) {
		// Si la condición se cumple como verdadera, entonces:
    echo "La clave 'clave1' existe en el array.";
} else {
		// Si la condición no se cumple como verdadera, entonces:
    echo "La clave 'clave1' no existe en el array.";
}

Verificación de variables de sesión:

session_start();
// Session_star Es una función en PHP que se utiliza para
// iniciar o reanudar una sesión.

// Verificamos si el ID del usuario existe en nuestra base de datos.
if (isset($_SESSION['usuario_id'])) {
		// Si el ID del usuario, existe, entonces
    echo "El usuario está autenticado.";
} else {
		// Si NO existe el ID del usuario, entonces
    echo "El usuario no está autenticado.";
}

empty.

  • empty se utiliza para comprobar si una variable está vacía. Una variable se considera vacía si no está definida, es null, o su valor es igual a cero o una cadena vacía.
  • Devuelve true si la variable está vacía, de lo contrario, devuelve false.
// Declaramos / definimos una variable a la que le asignaremos el valor de 0.
$edad = 0;

// Verificamos si la variable esta vacia o o no, con una condicion
// Si, la variable edad contiene algo, entonces.
if (empty($edad)) {
		// Imprime en la pantalla el siguiente mensaje, si la variable contiene algo.
    echo "La variable está vacía o tiene un valor igual a cero.";
} else {
		// Imprime en la pantalla el siguiente mensaje, si la variable es nula.
    echo "La variable no está vacía.";
}

// Hay que recordar que, una variable nula, en programación,
// generalmente se refiere a una variable que no tiene un valor
// asignado o que está vacía.

// $edad = 1; ---------------------------> No es una variable nula.
// $edad = "Hola mundo"; ----------------> No es una variable nula.
// $edad = function miFuncion(); --------> No es una variable nula.
// $edad = null; ------------------------> Es una variable nula.
// $edad; -------------------------------> Es una variable nula.

Funciones incorporadas..

Las funciones incorporadas en PHP son funciones que vienen predefinidas en el lenguaje y están disponibles para su uso sin necesidad de realizar ninguna importación o inclusión adicional. Estas funciones cubren una amplia variedad de tareas y facilitan la programación en PHP al proporcionar funcionalidades comunes.

Algunas categorías comunes de funciones incorporadas en PHP incluyen:

  1. Funciones de cadenas: Manipulación y manipulación de cadenas de texto (strlen, strpos, substr, etc.).
  1. Funciones matemáticas: Operaciones matemáticas básicas y avanzadas (sqrt, rand, round, etc.).
  1. Funciones de fecha y hora: Manipulación de fechas y horas (date, strtotime, time, etc.).
  1. Funciones de arrays: Operaciones en arrays (count, array_push, array_merge, etc.).
  1. Funciones de archivos: Lectura y escritura de archivos (file_get_contents, file_put_contents, etc.).
  1. Funciones de formulario y entrada de usuario: Manipulación de datos del formulario y entrada del usuario (isset, empty, htmlspecialchars, etc.).
  1. Funciones de base de datos: Interacción con bases de datos usando extensiones como MySQLi y PDO (mysqli_query, mysqli_fetch_assoc, PDO::query, etc.).
  1. Funciones de salida: Envío de datos al navegador (echo, print, var_dump, etc.).
  1. Funciones de red: Interacción con protocolos de red (file_get_contents para leer desde URL, curl para solicitudes HTTP avanzadas, etc.).

Estas son solo algunas categorías y ejemplos.
